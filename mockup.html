Share Button Feature

Location: Top right corner, next to the help button
Icon: ⎘ (copy/share icon)
Function: Copies current preset as URL to clipboard

When clicked:

  1. Generates URL with all current settings as query parameters
  2. Copies URL to clipboard using navigator.clipboard.writeText()
  3. Shows checkmark (✓) for 2 seconds as visual feedback
  4. Falls back to alert with URL if clipboard API fails

Parameters included:

  • All 15 sliders (factor, size, textsize, etc.)
  • All 5 checkboxes (audio reactive, filled circles, etc.)
  • Color palette selection
  • Built-in font selection (excludes custom uploaded fonts)
  • Text content
  • Pan and zoom values

Parameters excluded:

  • ❌ Uploaded audio files (stay on your computer)
  • ❌ Custom uploaded fonts (stay on your computer)
